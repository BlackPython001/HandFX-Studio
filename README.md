# HandFX-Studio


A browser-based real-time hand tracking visual experience built with MediaPipe Hands and HTML5 Canvas.

Neon Aura AR transforms hand movements into an interactive digital artwork featuring particles, glowing skeletons, dynamic lighting, gesture recognition, procedural backgrounds, multiple visual themes, and reactive audio.

Everything runs directly inside the browser without requiring installation or external software.

---

## Features

- Real-time hand tracking
- Supports up to 2 hands
- Neon glowing hand skeleton
- Dynamic fingertip particles
- Gesture recognition
- Pinch detection
- Shockwave effects
- Matrix-inspired animated background
- Five visual themes
- Audio-reactive humming engine
- Lightning between fingertips
- Procedural particle engine
- Motion-based background speed
- Ripple animations
- Glassmorphism interface
- FPS monitor
- Responsive fullscreen canvas

## Gesture Controls

### Open Hand

Displays glowing skeleton with fingertip particles.

---

### Pinch Gesture

Touch thumb and index finger together.

Triggers:

- Shockwave
- Particle burst
- Audio zap

---

### Two Hands

When both hands are visible:

- Dynamic gradient beams
- Lightning effects
- Rotating mandala connections
- Audio frequency modulation

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Neon-Aura-AR.git
```

Open the project

```bash
cd Neon-Aura-AR
```

Run using any local web server.

Example using VS Code Live Server.

Or with Python:

```bash
python -m http.server
```

Then visit

```
http://localhost:8000
```

Grant webcam permission and click **Enter Experience**.
